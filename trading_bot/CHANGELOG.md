\# Changelog



\## \[7.0] - Current

\### Added

\- 24시간 거래대금(Turnover) 필터 추가 (최소 10M USDT 이상)

\- 호가창 유동성 부족으로 인한 시장가 손절 슬리피지(Slippage) 방지 로직 구현



\## \[6.0]

\### Added

\- 멀티 타임프레임(MTF) 동적 타겟팅 구현: TP2를 4시간봉 볼린저 밴드 중앙선으로 변경하여 추세 추종 최적화

\- 수익 보존(Trailing Stop) 방어 로직 추가: TP1 체결 시 잔여 물량의 SL(Stop Loss)을 진입가와 TP1의 50% 지점으로 상향 조정하여 강제 수익 마감 보장



\## \[5.0]

\### Added

\- 손익비(Risk/Reward) 스크리닝 필터 추가: 예상 손실이 예상 수익의 1.5배를 초과할 경우 진입 차단

\- 포지션 진입 거절 사유에 대한 상세 로깅(Logging) 추가



\## \[4.0]

\### Added

\- Bybit V5 API의 Closed PnL 엔드포인트를 활용한 실시간 장부(CSV) 동기화 기능 추가

\- 수수료 및 펀딩비가 모두 반영된 정확한 Net PnL 추적 구현

\- 동적 지정가(Dynamic Limit) 매매 로직 추가



\## \[3.0]

\### Added

\- 백그라운드 실시간 포지션 및 호가창 모니터링 데몬(10초 주기) 추가



\## \[1.0] - \[2.0]

\### Added

\- 1시간/15분봉 기반 볼린저밴드 및 RSI 지표를 활용한 역추세(Short) 스나이핑 봇 초기 구조 설계

\- 로컬 JSON 기반의 포지션 상태(State) 관리 로직 구현
